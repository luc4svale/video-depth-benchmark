# Predictions

Raw model outputs plus visualization videos, one subfolder per model:

```
vda/            tum_*_depths.npz, tum_*_vis.mp4, tum_*_src.mp4
                 qualitative/bumba_meu_boi_vis.mp4, bumba_meu_boi_src.mp4

depthcrafter/   tum_*.npz, tum_*_vis.mp4
                 qualitative/bumba_meu_boi_vis.mp4

chronodepth/    tum_*_depth.npy, tum_*_depth.mp4, tum_*_depth_recolored.mp4
                 qualitative/bumba_meu_boi_depth.mp4, bumba_meu_boi_depth_recolored.mp4
```

Only the `.npz`/`.npy` files are read by `scripts/evaluate_all.py`. The
`.mp4` videos are for visual inspection and for the qualitative comparison
figure. No raw array is kept for the qualitative video — no ground truth
means it's not needed for anything but the visuals.

These are committed so `scripts/evaluate_all.py` reproduces the paper's
Table I / Table II numbers immediately after cloning, no GPU required.

Generated by `notebooks/02_infer_vda.ipynb`, `03_infer_depthcrafter.ipynb`
and `04_infer_chronodepth.ipynb`. Re-run those (GPU required) only to
regenerate predictions from scratch.
